Introduction

What is this guide?

This guide will introduce you to the Deck of Cards API. By the end of this guide you will be able to build a complete video poker application using just the API, JavaScript, and some HTML. The menu found in the top right corner works as a table of contents. It is available on every page. This guide was designed to be sequential, so please start at the top and work your way down each section, concluding with our final product, a video poker application! Before we get started, let us discuss some necessary background information, starting with what is an API?
