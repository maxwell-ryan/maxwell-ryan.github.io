JavaScript Object Notation (JSON)

An overview

I find it helpful to think of sheet music when trying to understand JSON and its purpose. In music, songs are written for musicians using a standardized notation. No matter what instrument you play, you can read all sheet music, since the notation is the same. JSON is very similar. JSON allows us to share inforation accross applications and networks using a standardized format. The name can be deceiving. Although JavaScript is in the name, JSON is used accross many languages. This way, when we receive information, we know exactly how to read it and use it in our own application, regardless of the application language used.

If you have not already, please take a look at the official JSON page.

Now that you have a better idea of JSON, let us take a look at an example response from the Deck of Cards API. All responses from the API are returned in JSON format.

This response is in JSON format. First, notice that everything is between brackets. Next, notice how each line represents a different name:value pair, seperated by a comma, except for the last pair. In JavaScript, which we will be using in this guide, these key:value pairs can be parsed directly into an object. Each key represents an object property, and the value is the property value. The first pair, would result in an object property of success and an associated boolean valu of true. Let's see JavaScript object representing the same data would look:

//first create an empty object, with undefined values
var responseObj = {
success: undefined,
deck_id: undefined,
shuffled: undefined,
remaining: undefined
};

//now assign object property values based on JSON response
responseObj.success = true;
responseObj.deck_id = 3p40paa87x90;
responseObj.shuffled = true;
responseObj.remaining = 52;

We now have a JavaScript object, that captures the information retruned by the API, which we can use in our own code/application! Notice how the final object looks nearly identical to the response:

//our object capturing the API JSON response
var responseObj = {
success: true,
deck_id: 3p40paa87x90,
shuffled: true,
remaining: 52
};

Now that you know about JSON and how to read JSON responses, continue here to start learning what methods the responses relate to, and how to interpret the response details. You will also see a trick to parse JSON into JavaScript objects more easily than the above.
