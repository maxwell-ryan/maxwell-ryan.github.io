Advanced deck creation technique

How to simplfy using the API and encorporate into our own functions

In our previous lesson, we learned how to create a deck using this JSFiddle. Please read these first if you have not already, then come back.

Before we continue, we should improve a few things from the the basic deck creation lesson to carry forward. We know how to create a new deck, but what if we created our own function which does all of this for us? And what if that function returned the deck_id?

Well this createDeck function does exactly that. It wraps all the code needed to create a deck into a single function, which returns our deck_id. Now creating a new deck is as simple as calling our own createDeck function. Give the result a try and see what the function returns.

Now we have a function that easily creates a deck for use in our own application, and returns a deck_id for use in future API calls.
